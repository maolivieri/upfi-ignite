## upfi - Ignite
Project created during the [@Rocketseat](https://www.rocketseat.com.br/) Ignite course as part of the chapter IV.
  
#### --- Features ---
- Upload images.
- Add a title.
- Host image on the cloud.
- View image album.
- Click to view image.
- Download image.
